MogDx with ChebNet

An upgraded AI-powered medical diagnosis and prediction platform integrating Chebyshev Graph Neural Networks (ChebNet) for advanced healthcare analytics and intelligent disease prediction.

Features :
Disease prediction using Machine Learning and Graph Neural Networks
ChebNet-based graph learning for improved relational data understanding
Data preprocessing and feature engineering pipeline
Interactive data visualization using Matplotlib
Efficient handling of healthcare datasets
Improved model explainability and prediction workflow
Modular and scalable project structure
Tech Stack
Python
ChebNet / Graph Neural Networks (GNN)
Pandas
NumPy
Matplotlib

Project Overview

MogDx leverages machine learning and graph-based neural architectures to analyze healthcare-related data and generate intelligent diagnostic predictions.
The integration of Chebyshev Graph Convolution (ChebNet) enables efficient learning from structured relationships within medical datasets, improving prediction capabilities and model performance.

Objectives
Build an intelligent healthcare prediction system
Explore graph neural networks in medical applications
Improve diagnostic accuracy using graph-based learning
Visualize and analyze patient-related data effectively
